iTunes Match

Load a Discogs folder and your iTunes XML. WaxFlow will match iTunes track metadata onto Discogs album tracks, then send only matched albums into Workflow Studio.
Matches artist + title + mix/remix wording. If one or more tracks on an album match BPM and/or Key, the whole Discogs album is sent to Workflow Studio. If none match, the album is skipped. Load All Tracklists fetches full Discogs release tracklists before matching with slower per-request pacing, retries, and local caching. Progress now shows a rough time estimate for tracklist loading, matching, and safe sending.

iTunes Scan

Load your iTunes XML, then scan or type a barcode. WaxFlow will pull the Discogs release, cross-reference the album tracks against iTunes by artist + title + remix wording, and automatically send the finished album to Workflow Studio if at least one track has BPM and/or Key.

Scan / Lookup

Album title does not need to match iTunes exactly. Matching is based on close artist, track, and remix wording. iTunes key is taken from the Key field or converted to Camelot from Comments just like the rest of WaxFlow.

WaxFlow Help

A quick guide to the main parts of the app. Use this tab as a reference while working.

Current tip: Start in Workflow Studio to import your library, clean metadata, and export a finished CSV.

📥 Import

Load XML, CSV, or TXT files into Workflow Studio. Drag and drop a file or use the browse buttons.

🎛️ Workflow Studio

Edit Artist, Song, Album, BPM, Key, Year, Genre, Artwork, Size, Speed, and Disc # directly in the table.

⌨️ Tab Empty Cells

When enabled, the Tab key jumps only to blank fields so cleanup goes much faster.

🛠️ Tools

Each row includes a compact Tools menu for Tunebat, Discogs web search, or a Discogs match.

🎨 Artwork / Discogs

Add your Discogs personal token to unlock artwork matching, release lookup, and scanner enrichment.

📷 Discogs Scanner

Scan or paste a UPC/EAN to pull release data, preview artwork, and send album tracks into Workflow Studio.

📤 Export

Export a clean CSV after editing. The file includes the hidden Artwork column along with your visible metadata.

⚡ Recommended Flow

Import your library, clean blanks, enrich with Discogs or scanner data, then export the finished CSV.
